# 非程式相關的開源授權

!!! info ""

    本篇翻譯自「[Non-Software Licenses | Choose a License](https://choosealicense.com/non-software/){target="_blank"}」。

開放原始碼軟體授權也可以用於非軟體的作品，而且往往是最佳選擇，特別是當相關作品能像原始碼一樣被編輯並進行版本控制時。請選擇一種開放原始碼授權。

## 資料、媒體

[CC0-1.0](https://choosealicense.com/licenses/cc0-1.0/){target="_blank"}、[CC-BY-4.0](https://choosealicense.com/licenses/cc-by-4.0/){target="_blank"} 與 [CC-BY-SA 4.0](https://choosealicense.com/licenses/cc-by-sa-4.0/){target="_blank"} 是常用於非軟體素材（從資料集到影片）的開放授權。請注意，Creative Commons（創用 CC）[不建議將其授權用在軟體或硬體上](https://creativecommons.org/faq/#can-i-apply-a-creative-commons-license-to-software){target="_blank"}。

## 文件

任何開放原始碼軟體授權或媒體的開放授權（同「資料、媒體」段落）也適用於軟體文件。若您的軟體與文件採用不同授權，請務必註明文件中的原始碼範例同樣依軟體的授權條款提供。

## 字型

SIL 開放字型授權（[SIL Open Font License](https://choosealicense.com/licenses/ofl-1.1/){target="_blank"}, OFL）1.1 能讓字型維持開放，並可自由用於其他作品中。

## 硬體

從客廳家具到 [FPGA](https://oshwa.org/resources/best-practices-for-sharing-fpga-designs/){target="_blank"}（現場可程式化邏輯閘陣列）的各式[開放硬體](https://oshwa.org/resources/open-source-hardware-definition/){target="_blank"}設計，可採用 CERN 開放硬體授權（CERN Open Hardware License）：[CERN-OHL-P-2.0](https://choosealicense.com/licenses/cern-ohl-p-2.0/){target="_blank"}（寬鬆授權）、[CERN-OHL-W-2.0](https://choosealicense.com/licenses/cern-ohl-w-2.0/){target="_blank"}（弱互惠）、[CERN-OHL-S-2.0](https://choosealicense.com/licenses/cern-ohl-s-2.0/){target="_blank"}（強互惠）。

## 混合式專案

若您的專案包含多種類型的素材，您可以同時使用多種授權，只要明確標示專案中每個部分適用的授權即可。可參考「[Choose a License](https://github.com/github/choosealicense.com#license){target="_blank"}」網站的授權聲明做為範例。
