為長期投入開源的你而設計。

在台灣，開源與開放社群一直都很活躍。有人寫程式、修 bug、有人撰寫文件、翻譯、設計、也有人投入社群經營、活動籌備，默默讓專案與社群能夠持續運作。

這些投入，未必總是顯眼，但正是它們，讓開源得以長久！OSCVPass 希望回應的，正是這樣的長期付出。

## 從一次性回饋，走向「開源貢獻者資格認證」

過去，OSCVPass 以研討會門票回饋為主要形式，鼓勵更多人踏出參與開源的第一步。隨著社群成長，我們也逐漸發現：單次的回饋，已不足以支持持續投入的人。

因此 OSCVPass 轉型為一個以「近一年實際開源貢獻」為基礎的資格認證機制：不是只為了一場活動，而是陪伴你一整年的開源旅程。

## 貢獻，不只是一種形式

OSCVPass 2.0 認可的開源貢獻，包含但不限於：

- 程式碼撰寫與維護
- 文件、翻譯、設計與非程式碼作品
- Issue 回報、討論參與
- 社群經營、志工投入、活動籌備
- 專案建立、授權選擇、文件整理與國際化準備

只要你在近一年內，實際投入於採用 FSF／OSI 認可授權的開源專案，並能透過公開方式被查證，這些付出都值得被看見與肯定。

## 一張 Pass，連結的是一個生態系

通過 OSCVPass 2.0 的申請，代表的是：

- 你被視為一位持續投入的開源貢獻者
- 你的付出，會被記錄、被理解、被分享
- 你將有機會在研討會、社群、專案與夥伴產生合作機會

OSCVPass 希望建立的，不是一對一的交換，而是一個讓貢獻能回流社群、再持續擴散的正向循環。

## 如果你正在做開源

OSCVPass 2.0 並不是在找「最厲害的人」，而是在找願意持續投入的人。

不論你的角色是工程師、文件撰寫者、設計師、社群組織者，只要你正在為開源付出，OSCVPass 都希望成為支持你走得更久的一部分。

邀請你，一起把開源這條路走得更長遠。
