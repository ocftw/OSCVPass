# 資料戰爭邁向數位化：Firefox 的卡牌遊戲現已上線

!!! note "引言"

    文章翻譯自《[Data War goes digital: Firefox’s card game is now online](https://blog.mozilla.org/en/firefox/bbo-data-war-digital-game/){target="_blank"} – Mozilla Blog》 CC-BY 4.0

上個月，Firefox 迎來 21 歲生日，象徵我們二十多年來持續打造一個展現創意、獨立性與信任感的網路世界。在 TwitchCon 上，我們用把億萬富翁送上太空，以及推出全新的卡牌遊戲《Data War》來一起慶祝。

「Billionaire Blast Off」的概念很簡單：把億萬富翁送上太空，然後樂在其中。而在《Data War》中，我們打造了一款既有趣又常常充滿混亂的遊戲，玩家要競爭贏得一張單程太空票，送一位對資料極度渴求的億萬富翁上太空。我們非常開心看到這麼多在 TwitchCon 的人玩得不亦樂乎。

你可以在這裡下載《Data War》的實體牌組，而現在，這場混亂也來到了你的瀏覽器中。《Data War》的數位版現已開放，免費在線上遊玩。

## 從展場現場到你的螢幕

在 TwitchCon 期間，參觀者把桌子擠得水洩不通，彼此對戰，高喊著「Data Grab！」，還在遊戲進行中交換牌組，看著億萬富翁一個個升空進入軌道。全新的線上版本，將這股同樣的熱鬧能量帶給每一個人。

「如果你能對某件事發笑，你就對它擁有力量，」Firefox《Data War》合作夥伴、Mondo Robot 執行創意總監 Dave Rowley 表示。「我們盡可能把荒謬感當作一種宣洩情緒的手法來運用。這讓我們能在『億萬富翁靠你的資料獲利』這樣的現實之中，加入一種去脈絡化的諷刺感，為挫折感找到出口，並透過真正有趣、容易上手的遊戲體驗，讓玩家重新奪回一點掌控權。」

## 它的起源與演變過程

Rowley 與 Firefox 團隊攜手合作，將《Data War》設計成一款一學就會、混亂無窮的遊戲。你可以把它想像成《[War](https://en.wikipedia.org/wiki/War_(card_game)){target="_blank"}》遇上《[Exploding Kittens](https://www.explodingkittens.com/){target="_blank"}》：資料就是貨幣、億萬富翁充滿不可預測性，而 Firefox 則會現身提醒玩家，真正掌握主導權的人其實是你。

為了把 Firefox 的觀點帶進遊戲創作過程，團隊也邀請實際打造 Firefox 的成員一同參與。其中一位是 Firefox 工程總監 Philip Luk，他和自己正值青少年的孩子們一起測試了實體遊戲的早期版本。他們的回饋，幫助《Data War》發展成一款比經典《War》更具動態性的遊戲。

「這款遊戲的目標，是突顯大型科技公司及其億萬富翁老闆如何從我們的資料中獲利」Firefox 工程總監 Philip Luk 表示。「我和孩子們也一起提出了新卡牌的點子，加入不同的策略變化，讓遊戲不只是翻牌而已，而是關於即時反應、大笑，還有看著混亂局面逐漸展開。」

Luk 也補充道：「和青少年一起進行遊戲測試，幫助我們看見哪些地方可以變得更不可預測、更有趣。那些出乎意料的瞬間，正是讓遊戲變得引人入勝的關鍵。」

在 TwitchCon 之後，團隊開始著手打造《Data War》的數位版本：一款更輕量、以瀏覽器為基礎的遊戲，保留混亂與幽默的精神，卻能隨時隨地用短短幾分鐘遊玩。

Mozilla 行銷合作與活動負責人 Benton Persons 表示：「我們希望設計一個任何人在需要短暫休息時，都能隨時遊玩的《Data War》數位版。這正是 Firefox 的精神所在：因為你掌控自己的使用體驗，讓上網變得不那麼有壓力。再說了，誰不想在會議空檔，把小小的億萬富翁送上太空呢？」

## 為樂趣而生，由價值驅動

每一場對戰，都是在提醒你，億萬富翁與大型科技公司為了從你的個人資料中獲利，會做出多麼荒謬的事情。

但同時，這也提醒玩家，真正掌握主導權的是你，最終把那些億萬富翁送上太空的人，也是你。

Rowley 表示：「我們的目標很簡單：讓《Data War》在瀏覽器裡和在桌上玩時一樣混亂、同樣好玩。因此我們簡化了規則，並加入只有數位版才有的元素，例如動畫、快速回合，以及具故事感的橋段，像是「Data Grab」小遊戲和「Billionaire Blast Off」的勝利演出，讓每一局即使是單人遊玩也能保持新鮮感。當整張桌子爆出笑聲的時候，你就知道你成功了。」
