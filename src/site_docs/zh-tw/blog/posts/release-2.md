# OSCVPass 2.0 正式發布，並同步推出「社群合作夥伴機制」

我們很高興向大家宣布，**[OSCVPass 2.0 已正式發布](../../v2/index.md)**，並同步推出全新的「**[社群合作夥伴機制](../../partner/index.md)**」，這是 OSCVPass 發展歷程中的一個重要里程碑。

OSCVPass 從台灣出發，長期關注並支持開源貢獻者。在 2.0 階段，我們正式轉型為一個以「**開源貢獻者資格認證**」為核心的計畫，目標是打造一個能夠**長期運作、可被信任、也能持續累積價值**的開源生態系，而不只是一次性的回饋或活動。

我們也觀察到，許多對開源有興趣的新手與潛在貢獻者，在真正踏出第一步前，往往需要：

- 一個安全的練習場域
- 清楚可理解的參與指引
- 有人陪伴、願意帶路的社群環境

<!-- more -->

因此，OSCVPass 2.0 同步推出「**社群合作夥伴機制**」，希望與各類型社群攜手合作，為參與者打造一條 「**可被引導、可被記錄、也可被認可**」 的開源參與路徑。

在這個合作機制中：

- OSCVPass 專注於制度設計、資格認證與生態系連結
- **社群夥伴**則專注於帶人、教學與創造實際的參與機會

OSCVPass 會協助明確定義哪些產出可作為貢獻認定、提供合作社群的對外標示、整理貢獻紀錄與證明方式，並在公開管道上呈現各社群的參與路徑，與社群一起累積可持續的開源參與模式。

我們誠摯邀請對開源教育、社群經營或培育新手貢獻者有熱情的團體與社群，一起加入 OSCVPass 的合作行列。讓更多人能在被支持、被看見、被認可的狀態下，踏上開源貢獻之路。
