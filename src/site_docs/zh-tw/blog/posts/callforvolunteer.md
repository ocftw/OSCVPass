# 【志工招募中】一起推動 OSCVPass 2.0，打造台灣可持續的開源參與路徑

你是否關心台灣的開源生態、社群發展，卻常覺得「想幫忙但不知道從哪開始」？OSCVPass 正在 [2.0 階段](../../v2/index.md)，正式轉型為 「**開源貢獻者資格認證**」計畫，我們非常需要專案志工加入，一起把這條路走得更穩、更長久。

## OSCVPass 在做什麼？

OSCVPass 是一個從台灣出發、長期支持開源貢獻者的計畫。我們專注在：

- 提供開源貢獻的制度化認可
- 協助社群整理與記錄可被承認的貢獻
- 串聯社群夥伴，打造**可被引導、可被記錄、也可被認可的參與路徑**

在這個分工中，OSCVPass 負責資格認證與生態系連結，而社群夥伴專注在帶人與創造參與機會。

## 我們在找這樣的「專案志工」

不需要你是開源大神，我們更在意你是否：

- 對開源、社群或非營利專案有興趣
- 願意一起討論、整理、推動事情
- 能穩定參與（每週或每兩週投入一些時間）

可能參與的事情包含（依你的興趣與時間調整）：

- 協助 OSCVPass 2.0 專案規劃與進度推進
- 與社群合作夥伴溝通、整理參與流程
- 協助文件、指引、對外說明內容的產出
- 一起思考如何讓開源貢獻「**更容易開始、也更容易被看見**」

加入你可以得到什麼？

- 實際參與台灣長期開源支持計畫的核心運作
- 與不同領域的開源與社群夥伴合作
- 累積可被看見的社群、專案與非營利經驗
- 親身參與打造「**不只是一次活動**」的開源生態系

> 我們不要求你「很厲害」，只希望你願意和我們（OCF）一起把事情慢慢做好！:slight_smile:

# 有興趣加入？

我們目前正在找尋以下類型的志工，一起加入專案推進：

- :technologist: 文件與制度整理志工（Documentation / Tech Writing）
- :technologist_tone1: 專案管理與營運協助志工（Project Management / Ops）
- :technologist_tone2: 社群與志工協調志工（Community / Volunteer Coordinator）
- :technologist_tone3: 內容與訪談志工（Content / Interview）
- :technologist_tone4: 視覺與設計志工（Design / Visual）

完整的志工招募可以參考[這篇說明](../../volunteer/index.md)，歡迎送出專案志工申請。也歡迎轉貼分享給你身邊關心開源與社群的朋友 :raised_hands:
