# :material-hand-wave: 志工招募

OSCVPass 是一個從台灣出發、長期支持開源貢獻者的計畫。

在 2.0 階段，我們希望從一次性的回饋機制，轉型為**可持續運作的開源支持生態系**，而這件事需要更多來自社群的夥伴一起完成！我們正在招募志工，一起協助 OSCVPass 在 2026 穩定運作、逐步成長。

## 志工參與方式

- 形式：遠端為主，非全職、非即時待命
- 投入時間：每週約 1 ~ 3 小時（可彈性調整）
- 建議期間：至少 3 個月
- 適合對象：
      - 有開源、社群、非營利、內容或專案經驗
      - 願意用自己的專業支持台灣開源生態的人

## 招募職缺一覽

### :technologist: 文件與制度整理志工（Documentation / Tech Writing）

- **角色說明：**協助整理、優化 OSCVPass 2.0 的說明文件，讓申請者、合作夥伴與社群能清楚理解計畫方向與參與方式。
- **你會協助的事情**
      - 改寫與優化申請說明文件
      - 協助區分「個人、社群、專案」三種申請類型的說明
      - 整理 FAQ、申請範例、非程式碼貢獻指引
      - 協助維護專案頁面與文件結構
- **我們期待你具備**
      - 喜歡整理資訊、把複雜事情講清楚
      - 基本文字編輯能力（Markdown 佳）
      - 有技術文件、社群文件或教學經驗加分

### :technologist_tone1: 專案管理與營運協助志工（Project Management / Ops）

- **角色說明：**協助 OSCVPass 在有限資源下，維持穩定節奏，讓事情「真的有往前走」。
- **你會協助的事情**
      - 協助拆解[季度](../v2/index.md#oscvpass-2026)、月度目標（Q1 ~ Q3）
      - 追蹤志工任務進度、整理待辦事項
      - 協助記錄會議重點與行動項目
      - 協助建立簡單、好維護的協作流程
- **我們期待你具備**
      - 對專案管理、流程優化有興趣
      - 熟悉 [GitHub Projects](https://github.com/orgs/ocftw/projects/3){target="_blank"} 或其他專案管理工具
      - 能主動提醒、整理與回饋

### :technologist_tone2: 社群與志工協調志工（Community / Volunteer Coordinator）

- **角色說明：**協助建立一個「**不燃燒志工**」的合作環境，讓參與的人能長期留下來。
- **你會協助的事情**
      - 協助志工招募與 onboarding
      - 定期關心志工參與狀況與回饋
      - 協助社群溝通與跨角色協調
      - 協助維護 Slack、郵件群組溝通空間
- **我們期待你具備**
      - 對社群經營、志工協作有興趣
      - 樂於傾聽、溝通與協調
      - 有社群或非營利經驗者加分

### :technologist_tone3: 內容與訪談志工（Content / Interview）

- **角色說明：**透過文字與訪談，累積台灣開源貢獻者與專案的故事，讓貢獻被看見。
- **你會協助的事情**
      - 規劃與撰寫貢獻者、專案訪談
      - 協助編輯社群投稿內容
      - 將開源經驗轉化為易讀的故事內容
- **我們期待你具備**
      - 喜歡寫作、採訪或內容整理
      - 能把技術或社群經驗轉成一般人看得懂的文字
      - 不需專業記者背景，有熱情即可

### :technologist_tone4: 視覺與設計志工（Design / Visual）

- **角色說明：**協助 OSCVPass 將制度與理念，用更好理解、更好分享的方式呈現。
- **你會協助的事情**
      - 設計申請流程圖、懶人包
      - 協助社群宣傳素材或簡報視覺
      - 提供視覺風格建議與範本
- **我們期待你具備**
      - 基礎平面或數位設計能力
      - 熟悉 Figma、Canva 等協作或開源工具（其一即可）
      - 願意與非設計背景的夥伴協作

## 為什麼加入 OSCVPass？

- 參與台灣長期開源支持計畫的核心運作
- 與不同領域的開源與社群夥伴合作
- 累積實際可被看見的社群與非營利經驗
- 一起打造可持續的開源生態系，而不只是一次活動

> 我們不要求你「很厲害」，只希望你願意和我們（OCF）一起把事情慢慢做好！:slight_smile:

## 如何加入

如果你對其中一個角色有興趣，或想用自己的方式參與，歡迎直接與我們聯繫（可附上簡單自我介紹）。請寄信到 <oscvpass@ocf.tw> 標題開頭註明 **[OSCVPass]** 內文包含簡單的自我介紹與您的想法，我們會儘快回覆，感謝！
